## 기도인도

일주일마다 1명씩 번갈아가면서 아침에 진행하는 매일합심기도 인도를 담당합니다. 기도제목은 선한목자교회의 [예수동행기도 사이트](http://www.praywithjesus.org/)에서 당일 **매일 합심기도**를 이용합니다.
